Program visualization

alpha test

Arguably, the single most useful static reflection call is [VisualizeGraph TaskCalls], which shows you a map of your program and what calls what.â€‚ For our treasure grammar, that gives us the diagram:

graph TD
n0("Treasure") --> n1("Weapon")
n1 --> n2("PreBuff")
n1 --> n3("WeaponType")
n1 --> n4("PostBuff")
n4 --> n5("Element")
n4 --> n6("Effect")
